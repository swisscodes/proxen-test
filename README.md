# Proxen

1. run npm i   / turkce npm install komutunu çalıştırın

2. create .env file with this variable DATABASE_URL=postgres://user:password@localhost:5432/db \
    bu değişkenle .env dosyası oluşturun DATABASE_URL=postgres://user:password@localhost:5432/db

3. npm run dev

4. postman kullanmadim bir klasor var klasor adi http incinde test.http var orda hazirlandim tum http requestlar \
    sadace manual olarak, alirsiniz token koyasirniz bir tane ornek birakacam unutmazsam.
